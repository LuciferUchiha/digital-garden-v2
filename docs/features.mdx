This page is to give an overview of what can be rendered on this website. For starters

Custom plantuml theme with procedures. TODO: SVGs so that can copy text.

```plantuml
@startuml
!theme purplerain from https://raw.githubusercontent.com/LuciferUchiha/georgerowlands.ch/main

Bob -> Alice :  normal
Bob <- Alice :  $success("success: Hi Bob")
Bob -x Alice :  $failure("failure")
Bob ->> Alice : $warning("warning")
Bob ->> Alice : $info("finished")

@enduml
```
